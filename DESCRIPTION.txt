Package: smap
Title: Nonlinear forecasting and dynamical system analysis
Description: smap is a set of tools for analyzing time-series data and
  forcasting into the future.  It is based on the methods described in
  Sugihara (1994)
URL: http://www.noamross.net
Version: 0.1
